# Frontend

## Testing
The application uses **Vitest** and **React Testing Library** for testing.

To run tests:
```bash
npm run test
```

To run specific file tests:
```bash
npm run test <file-name>
```

To run tests and generate a coverage report:
```bash
npm run test:coverage
```

## Code Formatting and Linting

This project uses **ESLint** to enforce code quality and formatting. 

To run the linter and check for issues:
```bash
npm run lint
```
